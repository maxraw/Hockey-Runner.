# Tuning notes

## Главные параметры в `cv_tracker_service.dart`

### Поле

- `area < frameArea * 0.12` — минимальная площадь поля.
- `aspectRatio < 1.2 || aspectRatio > 3.2` — допустимая форма поля.

Если поле не находится:
- уменьшай `0.12` до `0.08`;
- расширяй диапазон aspect ratio.

Если поле находится слишком часто по ложным объектам:
- поднимай `0.12` до `0.15`;
- сужай диапазон aspect ratio.

### Шайба

- `area < field.area * 0.001 || area > field.area * 0.08`
- `circularity < 0.35`
- `diameter < 8`

Если шайба пропадает:
- уменьши нижнюю границу площади;
- уменьши нижнюю границу circularity до `0.28`.

Если находятся ложные тёмные объекты:
- подними circularity;
- сузь допустимый размер.

### Управление

- jump: `dy <= -0.12`
- crouch: `dy >= 0.12`
- cooldown: `350 ms`

Если команда слишком чувствительная:
- сделай пороги `0.16` или `0.18`.

Если слишком вялая:
- опусти до `0.08`.

### Сглаживание

```dart
_lastSmoothed!.dx * 0.6 + current.dx * 0.4
_lastSmoothed!.dy * 0.6 + current.dy * 0.4
```

Больше первого коэффициента — движение плавнее, но с большей задержкой.
