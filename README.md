# Hockey Runner MVP

Первичный MVP для тестирования идеи:

- телефон-трекер с камерой;
- ручная калибровка 4 углов светлого поля;
- распознавание темной круглой шайбы диаметром около 7.5 см;
- фильтр от клюшки по форме, размеру, aspect-ratio и fill-ratio;
- экран игры на втором устройстве;
- локальная связка устройств через relay WebSocket на MacBook;
- старт игры через 15 секунд после кнопки «Начать игру».

## Что важно

Это тестовый пакет. Он специально сделан так, чтобы быстро проверить механику на iPhone 15 + MacBook/Xcode без написания нативного OpenCV-модуля. Камера и обработка кадра работают внутри WebView/Canvas. Для production-качества следующий шаг — заменить этот слой на нативный frame processor: iOS Swift/AVFoundation/OpenCV и Android Kotlin/CameraX/OpenCV или VisionCamera native plugin.

## Быстрый запуск

```bash
npm install
npx expo install --fix
npm run relay
```

Во втором терминале:

```bash
npm run prebuild:ios
open ios/*.xcworkspace
```

В Xcode выберите физический iPhone, Team в Signing & Capabilities и нажмите Run.

## Relay URL

После `npm run relay` терминал покажет адреса вида:

```text
ws://192.168.1.50:8787
```

Именно этот адрес нужно вписать в приложении на iPhone/iPad. `localhost` на физическом iPhone использовать нельзя, потому что это сам iPhone, а не MacBook.

## Первый сценарий теста

1. MacBook, iPhone и iPad/симулятор должны быть в одной Wi‑Fi сети.
2. На MacBook запустите `npm run relay`.
3. На iPhone откройте приложение, укажите Relay URL `ws://IP_MACBOOK:8787`, выберите «Открыть телефон‑трекер».
4. Нажмите «Включить камеру».
5. Наведите телефон сверху на светлое поле.
6. Нажмите 4 угла поля: верх‑лево, верх‑право, низ‑право, низ‑лево.
7. Положите темную шайбу на поле. Вокруг нее должен появиться зеленый круг.
8. На втором устройстве откройте это же приложение, тот же Relay URL и Room, выберите «Открыть экран игры».
9. Нажмите «Начать игру». Старт будет через 15 секунд.
10. Двигайте шайбу клюшкой: влево/вправо — уклонение, вперед — прыжок, назад — приседание.
